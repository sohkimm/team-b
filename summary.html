핵심 목표

주어진 두 NC 파일(CMEMS, SMAP)을 읽어 각 파일의 형태와 속성을 자동으로 파악하고, 같은 격자로 정합한 뒤 통계적으로 비교·검증한다. 결과는 지도 이미지와 산점도로 시각화한다.

NC 파일 구조 파악

변수명·좌표명·해상도·단위·시간 길이를 자동 탐지 → InspectReport

좌표 표준화 + AOI 클립

lat/lon → y/x 리네임, 경도 -180~180 정규화, 4D squeeze → 2D, 분석 영역 sel() 클립

격자 정합 (Resampling)

두 자료 중 한 쪽을 다른 쪽 격자로 맞춤 — coarse(다운샘플) 또는 fine(업샘플)

통계 계산

N, Bias (eval−ref), RMSE, MAE, R (Pearson), R²_NSE

시각화 저장

Step1·Step2 지도 이미지 (turbo colormap, 25~40 PSU) + 산점도 → results/figures/

Step 1 — 원본 해상도 지도

AOI 클립 후 각 자료 원래 해상도 그대로 시각화

CMEMS sos — 0.125°
매끄러운 그라디언트, 육지 마스킹(cfeature.LAND), 황해·동중국해 전반 36~37 PSU
SMAP smap_sss — 0.25°
블록 패턴(0.25° 격자), 흰색=NaN(해안 미관측), 32°N 122°E 양쯔강 유출 저염 (~24 PSU)

주요 관찰

  • CMEMS는 황해 전체에 걸쳐 비교적 균일하게 높은 염분 (36~37 PSU) — 블렌디드 제품 특성상 담수 유입 영향을 과소평가
  • SMAP은 양쯔강·황하 담수 유출로 인한 저염(24~28 PSU) 신호를 선명하게 포착
  • 두 자료의 공간 패턴은 유사하나 절댓값 차이 존재 → Bias 발생의 원인

Coarse(권장) vs Fine(참고용) 두 방식 모두 실행

✅ Coarse 모드 — CMEMS(0.125°)를 SMAP(0.25°) 격자로 블록평균 다운샘플

CMEMS → 0.25° (coarsen)
블록평균으로 다운샘플. 독립 관측 유지.

⚠️ Fine 모드 — SMAP(0.25°)를 CMEMS(0.125°) 격자로 bilinear 업샘플

SMAP → 0.125° (bilinear 업샘플)
매끄러워졌지만 새로운 해양 정보 없음
CMEMS 0.125° — ref (변환 없음)

Bias +1.1 PSU의 의미

  • CMEMS(eval)이 SMAP(ref)보다 평균 1.1 PSU 높게 나타남
  • CMEMS는 블렌디드 NRT 제품으로 황해·동중국해의 담수 유입(양쯔강·황하) 영향을 과소평가하는 경향
  • SMAP은 저염 신호를 더 직접적으로 포착 → 두 제품의 물리적 차이가 반영된 결과

✅ 업샘플이 맞는 경우

목적: 시각화, 스무딩, 디스플레이
TV 4K 업스케일, 구글맵 줌인, 기상 예보 영상 표시.
더 보기 좋게 만드는 것이 목표 — 정확도 무관.

❌ 업샘플이 틀린 경우

목적: 통계 검증, 두 자료 비교
업샘플된 격자점을 독립 관측으로 취급하면 N 과대, RMSE·R 왜곡.
Fine 모드 R 0.845는 실제 정확도 향상이 아님.

3가지 근본 원인

  • 신호처리 (Nyquist): 저해상도 샘플링에서 소실된 고주파 공간 정보는 업샘플로 복원 불가
  • 정보이론: 결정론적 보간(bilinear)은 정보(entropy)를 증가시킬 수 없다 — NCAR 명시
  • 통계학 (Pseudo-replication): N이 4배 늘어도 유효 자유도는 그대로 — Hurlbert (1984)
